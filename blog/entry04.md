#### Aframe
For CSS grid I learned how to make different 3d models using the website [aframe](https://aframe.io/). I tried adding different primitives like circles, boxes, texts, cylinder, traingles, and many more. I made a simple cat using three spheres and two traingles.
Example:
```bash
    <a-scene>
<a-sphere position="0 1.5 -5" radius="1" color="orange"></a-sphere>
<a-circle position="-0.3 1.7 -5" radius="0.1" color="black"></a-circle>
<a-circle position="0.3 1.7 -5" radius="0.1" color="black"></a-circle>
<a-cone position="-0.6 2.4 -5" radius-bottom="0.3" height="0.6" color="white"></a-cone>
<a-cone position="0.6 2.4 -5" radius-bottom="0.3" height="0.6" color="white"></a-cone>
<a-sky color="black"></a-sky>
    </a-scene>
```
I created this by using the starter example from the aframe website. I messed around with it and added some primitives. I then kept on adjusting everything until it places on the way I wanted it to be. For the positioning it is ```position= "x,y,z"``` The ```x``` is going left and right, the ```y``` is going up and down and the ```z``` is going backwards and forwards. The ```radius``` is how large the model is going to be. The ```radius-bottom``` is adjusting the width.
